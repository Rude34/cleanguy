description: "Fix BSOD issues appearing right after a Windows update with safe rollback and driver checks."
category: "Windows"

updatedAt: "2026-03-20"
---

If your PC started showing blue screen errors after an update, try these steps:

1. Boot into Safe Mode.
2. Open Settings > Windows Update > Update history.
3. Uninstall the most recent quality update.
4. Restart and test system stability.
5. Update chipset and display drivers from manufacturer website.

If it continues, run Startup Repair from Advanced startup options.
