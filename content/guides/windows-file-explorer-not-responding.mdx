---

When File Explorer hangs or crashes repeatedly:

1. Restart File Explorer from Task Manager.
2. Clear File Explorer history from Folder Options.
3. Disable unnecessary third-party shell extensions.
4. Run: `sfc /scannow`
5. Run: `chkdsk /scan`

Restart the PC and test by opening large folders and network paths.
