When disk usage stays at 100%, system response becomes very slow.

1. Open Task Manager and check Disk column.
2. Pause heavy background downloads/sync apps.
3. Temporarily stop SysMain service and compare usage.
4. Reduce Windows Search indexing scope.
5. Check disk errors with `chkdsk /scan`.

If using HDD, moving system to SSD gives major performance improvement.
