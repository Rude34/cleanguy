## Symptom
Wi-Fi appears connected, but websites do not load.

## Quick Checks
- Restart the modem/router
- Toggle Airplane mode on/off
- Confirm VPN/Proxy is disabled

## Commands
```powershell
ipconfig /flushdns
ipconfig /release
ipconfig /renew
```

## If Issue Persists
- Set DNS manually to `8.8.8.8` and `8.8.4.4`
- Update the network adapter driver from Device Manager
- Restart your PC and test again
