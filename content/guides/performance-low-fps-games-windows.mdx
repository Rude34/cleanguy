If games stutter or FPS drops suddenly:

1. Update GPU driver from NVIDIA/AMD/Intel official app.
2. Set Windows Power Mode to Best Performance.
3. Disable background recording overlays.
4. Lower in-game shadows/effects first.
5. Use fullscreen mode and test again.

Monitor CPU/GPU temperatures; thermal throttling can reduce FPS heavily.
