---
title: "Discord Mic No Input on Windows"
description: "Fix Discord microphone detection and voice input issues with permission and device selection checks."
category: "Apps"
minutes: 5
updatedAt: "2026-03-20"
---

If Discord shows no microphone input:

1. Allow microphone in Windows Privacy settings.
2. In Discord Voice & Video, select the correct Input Device.
3. Disable Automatic Input Sensitivity and set manually.
4. Run Discord as Administrator.
5. Disable Exclusive Mode for microphone in Sound settings.

Restart Discord and perform the built-in mic test.
