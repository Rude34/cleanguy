---

When Microsoft Store does not download or open:

1. Press Win + R and run `wsreset.exe`.
2. Go to Settings > Apps > Installed apps > Microsoft Store.
3. Use Repair, then Reset.
4. Sign out and sign in to Store again.
5. Check date/time and region settings.

Install pending Windows updates and test Store downloads again.
