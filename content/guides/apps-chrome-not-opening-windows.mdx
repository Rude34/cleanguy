---
title: "Google Chrome Not Opening on Windows"
description: "Fix Chrome launch issues by clearing cache files, disabling extensions, and checking user profile data."
category: "Apps"
minutes: 5
updatedAt: "2026-03-20"
---

If Chrome does not open or closes immediately:

1. End all Chrome processes from Task Manager.
2. Rename Chrome user data folder temporarily.
3. Launch Chrome in safe mode with extensions disabled.
4. Reinstall Chrome from official source.
5. Restart Windows and test.

If it only fails on one account, create a fresh browser profile.
